# SoulPhrase
> 마음을 울리는 한 구절에 끌려서
> 
> 책을 사거나, 영화를 보거나, 드라마를 정주행한 경험이 있나요?
### 감명 깊은 글귀를 기록 및 공유할 수 있는 인생 글귀 웹사이트

## A. 개발 팀원
- Front-end Developer [이서진](https://github.com/metamong-Hi)
- Back-end Developer [최윤정](https://github.com/letmeloveyou82)

## B. 개발 환경
- Front-end : JavaScript, React

- Back-end : Java, Spring Boot, Spring Security, Spring Data JPA

- Cloud : AWS RDS(MariaDB), AWS EC2
- IDE : Visual Studio Code, IntelliJ
